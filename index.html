HTML Tags

10 Basic HTML Tags

Tag Name Description
<html> The root element Contains all code. Specifies that this is an html document.
<head> The document head Contains information about the web page.
<title> The page title Contains the title of the page.
<body> The page's content Appears after the head element in the page, should contain all the page's content.
<h1> A section heading Headings let you break up your page content into readable chunks.
<p> A paragraph The p element lets you create paragraphs of text.
<a> A link The a element lets you create links to other content.
<img> An image The img element lets you insert images into your web pages.
<div> A block-level container for content Generic container used to add more structure to your page content.
<span> An inline container for content Used to add structure to your content but adds inline content.
